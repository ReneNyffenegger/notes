x86

Real mode

Real mode is supported on all x86s (starting with 8086?).
x86s start in real mode.

Segmentation

8086: 20-bit address, but only 16-bit registers. In order to make entire address space available, segmentation is used.
Memory diveded into 64 KB (16-bit) chunks («segments»). Start address of segment: segment-selector * 16.

Protected mode

32-bit address bus instead of 20-bit.
Paging

Segmentation

No fixed 64 KB chungs anymore, instead: size and location of segements described by segment descriptor (which are stored in the GDT - Global Descriptor Table).

64 bit mode

Some features of 64 bit:
  • 8 new general purpose registers: r8 .. r15
  • All general purpose registers are 64 bit.
  • 64 bit instruction pointer: RIP
  • New operatoing mode : long mode
  • 64-bit addresses and operands
  • RIP relative addressing

Long mode

Long mode is an extension of protected mode.
It consists of two sub modes
  • 64 bit mode
  • compatibility mode.
