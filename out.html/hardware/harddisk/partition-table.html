partition table

The partition table records a block device's division into partitions.
Partition tables can be created and manipulated with fdisk, parted on Linux and diskpart on Windows.
The partition table traditionally used on PC BIOS - Basic Input/Output System platforms is the MBR - Master Boot Record. But see also UEFI - Unified Extensible Firmware Interface.
