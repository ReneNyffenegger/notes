NetCDF - Network Common Data Format

Ein binäres, offenes Dateiformat für den Austausch von (hauptsächlich Klima- u. GEoinformations-) Daten.
Daten sind in einem Array abgelegt.
NetCDF Daten können zB mit QGIS angeschaut werden.
NetCDF basiert auf CDF - Common Data Format
