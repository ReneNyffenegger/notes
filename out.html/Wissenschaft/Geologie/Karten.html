SRTM

NASA Shuttle Radar Topographic Mission (SRTM): DEMs (Digital Elevation Data) für 80% der Erdoberfläche.
Daten werden zur Zeit von USGS unentgeltlich zur Verfügung gestellt (mittels des National Map Seamless Data Distribution System oder der USGS FTP Seite).
SRTM Daten kommen in zwei Genauigkeitsstufen: SRTM1 (USA, ihre Territorien und Besitztümer) in einer Auflösung von 1°x1° (ca 30 Meter) und SRTM3 in einer Auflösung von 3°x3° (ca 90 Meter).
Die Daten sinz zwischen 56° S und 60° N vorhanden. Es enthällt allerdings »Löcher«, die mittels Interpolation befüllt werden müssen.
Die Daten können von http://earthexplorer.usgs.gov/ heruntergeladen werden.
sa: https://github.com/ReneNyffenegger/about-GMT/blob/master/SRTM/earthexplorer.usgs.gov-zip_2_bil_2_nc.pl
Der vertikale Fehler betrage angeblich weniger wie 16 Meter.
Download: hgt Dateien von http://dds.cr.usgs.gov/srtm/version2_1/
http://srtm.csi.cgiar.org/ Downloads von 5°x5° Teilen von http://srtm.csi.cgiar.org/SELECTION/inputCoord.asp.
-> See also Earth2014

GMRT

Global Multi-Resolution Topography Data Synthesis
Ein DEM aus mehreren Quellen:
  • Ship-based multibeam swath bathymetry data (MGDS)
  • NED
  • GEBCO
  • IBCAO
  • IBSAO
Daten können über einen Web Service bezogen werden.
Folgende Formate stehen zur Verfügung
  • GMT3 NetCDF
  • COARDS konforme NetCDF
  • ESRI ASCII
  • GeoTIFF
http://onlinelibrary.wiley.com/doi/10.1029/2008GC002332/full

GSHHG

A Global Self-consistent, Hierarchical, High-resolution Geography Database. It amalgates two public domain databases: WVS (World Vector Shorelines) and WDBII (CIA World Bank II).
As of version 2.2.2, GSHHG is released under the GNU Lesser General Public license.
The data is obviously used by General Mapping Tools .
link: https://www.ngdc.noaa.gov/mgg/shorelines/gshhs.html
