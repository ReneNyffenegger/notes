keymap

A keymap is a keyboard table description.
The format is vaguely similar to those accepted by xmodmap.
The kernel has a keymap built into it. This keymap can be modified with loadkeys and viewed with dumpkeys.
The kernel source contains a default keymap: drivers/tty/vt/defkeymap.map.

Keycode definition

keycode keynumber = keysym keysym keysym ...
keynumber is an internal identification number for a key, roughly equivalent to its scancode.
