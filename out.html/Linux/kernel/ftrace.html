ftrace

ftrace = Function Tracer
ftrace uses the debugfs filesystem to hold the control files as well as the files to display output.

Common uses

Event tracing

Prerequisites

CONFIG_DYNAMIC_FTRACE, CONFIG_TRACE_IRQFLAGS_SUPPORT
