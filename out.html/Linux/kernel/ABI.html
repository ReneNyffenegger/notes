ABI - Application Binary Interface

The ABI «lies» between the user space and the kernel space.
Contrast with API (Application Programming Interface).
For different levels of ABI stability:
  • stable (Documentation/ABI/stable)
  • testing (Documentation/ABI/testing)
  • obsolete (Documentation/ABI/obsolete)
  • removed (Documentation/ABI/removed)
