kernel/uid16.c

This file defines the entry point for the following syscalls: chown16, lchown16, fchown16, setregid16, setgid16, setreuid16, setuid16, setresuid16, getresuid16, setresgid16, getresgid16, setfsuid16, setfsgid16, getgroups16, setgroups16, getuid16, geteuid16, getgid16, getegid16.
