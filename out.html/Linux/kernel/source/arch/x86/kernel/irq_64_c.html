arch/x86/kernel/irq_64.c

Contains the lowest level x86_64-specific interrupt entry and irq statistics code.
the remaining irq logic is done by the generic kernel/irq* code and in the x86_64-specific irq controllor code (e.g arch/x86/kernel/i8259.c, arch/x86/kernel/apic/io_apic.c)
