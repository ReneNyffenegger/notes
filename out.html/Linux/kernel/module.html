Linux kernel module

A (loadable) kernel module extends the functionality of the kernel (new hardware, Filesystem, system calls) or modifies the behavior of the kernel.
Modules can be loaded/unloaded without restarting the kernel.

Parameters

When loaded into the kernel, a module shows its parameters in /sys/module/module-name/parameter/parameter-name.

Module names

Hyphens ( -) and underscores ( _) are not differentiated in module names.
