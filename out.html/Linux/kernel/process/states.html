Linux process states

A linux process is in one of the following states:
  • Running or runnable (on the run queue)
  • Uninterruptible sleep (waiting for some event)
  • Interruptible sleep (wating for an event or or a signal)
  • Stopped (either by job control signal or because of being traced by a debugger)
  • Zombie process (terminated, but not reaped by parent process)
