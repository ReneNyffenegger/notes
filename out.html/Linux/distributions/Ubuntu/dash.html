Turning off sending data to productsearch.ubuntu.com

Entering a search term into the dash causes the keystrokes to be sent to productsearch.ubuntu.com and «selected third parties».
This can be turned of using the privacy panel (System Settings -> Security Settings -> Security & Privacy -> Search) and toggling off include online search results.
