/etc/passwd

User account information: each line represents a user and is colon-seperated with the following fields: 1) username, 2) password [nowadays usually x, see /etc/shadow ] 3) user id 4) default group id [See /etc/group] 5) gecos, 6) The user's home directory, 7) The name of the program to be run when a user logs in, usually a shell.
In order to prohibit a user from logging in, the programm to be launched (7) might be set to /bin/false
