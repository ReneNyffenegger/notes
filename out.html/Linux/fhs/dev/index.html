Search notes:

/dev

All device files should be located in the /dev directory.
/dev/hdX: IDE - Integrated Drive Electronics disks. /dev/hda represents the first IDE drive in the system.
/dev/sdX: sd stands for SCSI - Small Computer System Interface disk.
/dev/srX: *r: optical devices (cd rom)
