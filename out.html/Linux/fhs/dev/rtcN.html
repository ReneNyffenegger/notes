/dev/rtcN

While /dev/etc reports real time clock signals for ACPI - Advanced Configuration and Power Interface and PC - Personal Computer platforms, /dev/rtcN (where N is a number) supports other real time clock models.

Links

/sys/class/rtc/rtcN, /dev/etc
