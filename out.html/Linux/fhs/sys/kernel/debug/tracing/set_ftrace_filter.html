/sys/kernel/debug/tracing/set_ftrace_filter

Echo function name in here to only trace these functions. 
func_full_name
*end_of_func_name
*start_of_func_name
*middle_of_func_name*
:mod:name-of-a-module

Triggers

func_name:trigger[:count]
trigger on of traceon, traceoff, enable_event:<system>:<event>, disable_event:<system>:<event>, stacktrace, snapshot, dump, cpudump.
