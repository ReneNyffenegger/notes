/boot/grub/grub.cfg

The configuration file for GRUB - Grand Unified Bootloader (usually located under /boot/grub).
Created by Linux/shell/commands/grub-mkconfig.
This file controls the operation of grub-mkconfig.
It should be a valid POSIX shell script because it is sourced.
Usually, it contains just KEY=value lines.
Valid keys are
  • GRUB_DEFAULT
  • GRUB_SAVEDEFAULT
  • GRUB_TIMEOUT
  • GRUB_HIDDEN_TIMEOUT
  • GRUB_HIDDEN_TIMEOUT_QUIET
  • GRUB_DEFAULT_BUTTON
  • GRUB_TIMEOUT_BUTTON
  • GRUB_HIDDEN_TIMOOUT_BUTTON
  • GRUB_BUTTON_CMOS_ADDRESS
  • GRUB_DISTRIBUTOR
  • GRUB_TERMINAL (might be set to console to disable graphical terminal)
  • GRUB_TERMINAL_INPUT
  • GRUB_TERMINAL_OUTPUT
  • GRUB_SERIAL_COMMAND
  • GRUB_CMDLINE_LINUX
  • GRUB_CMDLINE_LINUX_DEFAULT
  • GRUB_CMDLINE_NETBSD
  • GRUB_CMDLINE_NETBSD_DEFAULT
  • GRUB_CMDLINE_GNUMACH
  • GRUB_CMDLINE_XEN
  • GRUB_CMDLINE_XEN_DEFAULT
  • GRUB_CMDLINE_LINUX_XEN_REPLACE
  • GRUB_CMDLINE_LINUX_XEN_REPLACE_DEFAULT
  • GRUB_CMDLINE_LINUX_DEFAULT
  • GRUB_DISABLE_LINUX_UUID ( true: Don't pass root=UUID=xxx parameter to Linux)
  • GRUB_DISABLE_RECOVERY
  • GRUB_VIDEO_BACKEND
  • GRUB_GFXMODE (Might be set to 640x480)
  • GRUB_BACKGROUND
  • GRUB_THEME
  • GRUB_GFXPAYLOAD_LINUX
  • GRUB_DISABLE_OS_PROBER
  • GRUB_INIT_TUNE ( 480 440 1: sound a beep at grub start)
  • GRUB_BADRAM
  • GRUB_PRELOAD_MODULES
For full documentation of these options see info -f grub -n 'Simple configuration'.
