GRUB - Grand Unified Bootloader

GRUB can not only load Linux but other Operating Systems as well.
Arguably the most flexible boot loader for Linux. Other Linux boot loader are SYSLINUX and LILO.
image directory (s.a. GRUB image files).

Installing GRUB

GRUB is installed with grub-install (grub-install), for example grub-install /dev/hda.

Two boot methods

GRUB boots either
  • the OS directly, or
  • it uses chain-load which loads another Linux boot loaders.

Windows

GRUB cannot directly boot Windows, it needs the chain-load mechanism.
