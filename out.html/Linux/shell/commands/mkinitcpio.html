mkinitcpio

Used to create an initial ramdisk environment, needed to boot the Linux kernel.

Some flags

-p: use a Preset.
-g: generate file with name specified.
-k: Use specified kernel version instead of current running kernel. Can be:
  • path to path to kernel image
  • kernel version
  • keyword none (which adds no modules to the image.

See also

initial ram disk, /etc/mkinitcpio.conf, /etc/initcpio/hooks, /etc/initcpio/install.
