useradd

Create a new user or update default new user information.
It is a low level utility. Therefore, you might want to use adduser.
Use the -m flag to create the new user's home directory if it does not exist.
If not invoked with -s to specify the shell, it assumes the value of SHELL variable in /etc/default/useradd.
If not invoked with -k, the files from the SKEL variable in /etc/default/useradd (default: /etc/skel/*) are copied into the new user's home directory.
A user is removed/deleted with userdel. 
useradd -m rene
passwd rene
