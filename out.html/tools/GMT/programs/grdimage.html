grdimage

grdimage is used to create color images.

Main options

The program takes the usual command options. The -R option is, by default, taken from the data set.
Other important options are
-E: Set desired resolution of image (default: data resolution).
-I: Use artificial illumination using intensities from indicated file.
-M: Force gray shading using the (television) YIQ conversion.
