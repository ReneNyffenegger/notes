grdraster.info

This file contains information on rasterfiles which are readable with grdraster. 
file_number "title string" "z units" -R -I GorP type scale offset NaNflag filename [B|L]
B = Big, L = Little Endian
This file needs to be updated when new data is isntalled on a system.
