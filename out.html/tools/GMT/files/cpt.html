GMT: CPT files

CPT = Color Palette Tables
A CPT file determines with what color a specific z value shuld be colored, for example in grdimage.
cpt files can be created with makecpt, grd2cpt or »manually« (for example using awk etc).
The cpt file is discussed in detail in the GMT Technical Reference and Cookbook, Chapter 4.

Built in CPTs

A list of built-in color tables can be viewed by just typing makecpt.
