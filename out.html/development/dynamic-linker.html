dynamic linker

Also known as: dynamic loader

/lib/ld-linux.so.2, ld-linux-x86-64.so.2
/lib/ld-linux.so.2.
The dynamic loader finds and loads shared libraries needed by a program when the program is run.
Determine the name of the dynamic loader with readelf -l executable | grep interpreter, for example readelf -l $(which ls) | grep interpreter.
The dynamic loader must not be confused with the standard-linker ( ld).
