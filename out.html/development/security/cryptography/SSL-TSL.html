SSL/TSL

SSL - Secure Sockets Layer and development/security/cryptography/TLS provide authentication and encryption over insecure networks.

History

SSL is called TLS - Transport Layer Security since version 3. (SSL 1.0 -> SSL 2.0 -> SSL 3.0 -> TLS 1.0 -> TLS 1.1 -> TLS 1.2). TLS 1.0 sometimes called SSL 3.1.
