JET

JET is an SQL database, developed my Microsoft, mostly for Access.
An (or the?) interface to JET is DAO - Data Access Objects.
By default, JET stores the files in MDBs. It also supports dBASE, Microsoft Fox Pro, Paradox, Excel and ODBC.
An alternative and successor to JET is MSDE - Microsoft Desktop Engine.
JET is a modified form of ISAM - Indexed Sequential Access Method.
