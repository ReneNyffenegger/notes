DAO - Data Access Objects

Workspaces

A workspace manages a session with the Microsoft Jet database engine. They reside in memory during such a session.
A Workspace consists of one or more open DAO database objects.
Often, they're used for transactional purposes.

Default Workspace

The first workspace is the default workspace.
