.git/HEAD

A symref to the .git/refs/heads namespace, describing the currently active development/version-control-systems/git/branch[branch].
HEAD is a ref [git].
Git only changes the content of this file when the branches is changed.

Content of HEAD

The Content of HEAD might be 
ref: refs/heads/branch_name.

detatched HEAD

If HEAD references a commit directly, it is called a detached HEAD. ( man git-checkout)
Normally the HEAD stores the name of a branch and commands that operate on the history HEAD represents operate on the history leading to the thip of the branch the HEAD points at.
However, Git also allows to check out an arbitrary commit that isn't necessarily the tip of any particular branch. In such a state, the HEAD is called «detached».
